# Preface {.unnumbered} This is the website for the **Introduction to Data Science**. The website for **Advanced Data Science** is [here](http://rafalab.dfci.harvard.edu/dsbook-part-2/). We make announcements related to the book on Twitter. For updates follow [\@rafalab](https://twitter.com/rafalab). This book started out as the class notes used in the HarvardX [Data Science Series](https://www.edx.org/professional-certificate/harvardx-data-science)[^index-1].
